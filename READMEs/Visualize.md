# Visualize Module

The Visualize module provides nodes for visualizing tensor data in various formats, making it easier to understand and debug deep learning models.

## Features

### Image Visualization
- TensorToImageVisualizer: Convert tensors to images for visualization
- TensorHeatmapVisualizer: Generate heatmap visualizations of tensor data

### Data Information
- TensorShapeVisualizer: Display tensor shape information
- TensorLineChartVisualizer: Create line charts from tensor data
- TensorStatisticsVisualizer: Compute and display tensor statistics (mean, std, min, max, etc.)

## Example Workflow

See `example_workflow.json` in the Visualize directory for a demonstration of how to use these nodes to visualize different types of tensor data.
